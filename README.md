# Instagram Clone

Instagram Clone is a simple Android social media application that allows users to create accounts, upload photos, and share posts with other users.

This project was developed to practice Firebase integration and understand the fundamentals of building a social media application on Android.

---

## 🚀 Features

- User Registration & Login
- Firebase Authentication
- Upload Photo Posts
- Add Post Description
- View Feed
- Firebase Storage Integration
- Real-time Post Listing

---

## 🛠️ Technologies Used

- Java / Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore Database
- Firebase Storage
- Material Design Components

---

## 📌 Project Purpose

The purpose of this project is to understand how a social media application works behind the scenes and gain hands-on experience with Firebase services such as authentication, storage, and database management.

# 🇹🇷 Türkçe

Instagram Clone, kullanıcıların hesap oluşturabildiği, fotoğraf yükleyebildiği ve gönderi paylaşabildiği basit bir Android sosyal medya uygulamasıdır.

Bu proje Firebase servislerini öğrenmek ve Android üzerinde bir sosyal medya uygulamasının temel mantığını anlamak amacıyla geliştirilmiştir.
